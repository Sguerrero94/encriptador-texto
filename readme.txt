# Encriptador y Desencriptador de Texto

Este proyecto es una aplicación web simple para encriptar y desencriptar texto utilizando HTML, CSS y JavaScript. La aplicación permite a los usuarios ingresar texto, encriptarlo, desencriptarlo y copiar el resultado.

## Estructura del Proyecto

- **`index.html`**: El archivo HTML principal que contiene la estructura de la página.
- **`styles.css`**: El archivo CSS para el estilo y diseño de la página.
- **`script.js`**: El archivo JavaScript que contiene la lógica para encriptar, desencriptar y copiar el texto.
